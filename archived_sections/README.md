## Notes

These sections were archived on 2025-11-14 to:
- Reduce clutter in the main shared/sections folder
- Keep ZIP file sizes smaller
- Improve build performance
- Make it easier to find actively used sections

## To Restore a Section

If you need to use any of these sections again:

1. Copy the `.liquid` file back to `/shared/sections/`
2. Re-export the theme
3. Upload to Kajabi

## Why These Were Archived

These are full-page sections that were generated for specific use cases but are not being actively used in the current themes. They can be recreated or customized when needed.
