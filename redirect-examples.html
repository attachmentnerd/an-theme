Common Redirect Scenarios

1. Old Course URL ‚Üí New Product URL

EXACT MATCH 
{ from: '/courses/secure-parenting', to: '/products/secure-parenting-program' }

2. Checkout Redirect to Specific Offer

CHECKOUT

3. Limited-Time Sale Redirect

TEMPORARY

  from: '/black-friday',

  permanent: false,

5. Mobile App Redirect

CONDITIONAL 
{

  to: '/mobile-download',
  condition: 'mobile'
}

Complete Implementation Example

<script>
(function() {
  const redirects = [
    // Redirect old course URLs
    { from: '/courses/secure-parenting', to: '/products/secure-parenting-program' },
    { from: '/course/attachment', to: '/products/attachment-course' },
    
    // Checkout redirects
    { from: '/checkout', to: '/offers/iWQFFjpH/checkout' },
    { from: '/pay-what-you-can', to: '/offers/aLCCMYCf' },
    
    // Sale/promo redirects
    { from: '/sale', to: '/offers/launch-special' },
    { from: '/50off', to: '/offers/half-price' },
    
    // Old blog redirects (pattern matching)
    { from: '/blog/old-', to: '/blog/', type: 'starts' },
  ];
  
  const currentPath = window.location.pathname;
  
  redirects.forEach(rule => {
    if (currentPath === rule.from || 
        (rule.type === 'starts' && currentPath.startsWith(rule.from))) {
      window.location.replace(rule.to);
    }
  });
})();
</script>

Testing Your Redirects

  1. Add the redirect rules to redirect_manager.liquid
  2. Include the snippet in your theme layout
  3. Visit the old URL to test the redirect
  4. Check browser console for any errors

Debug Mode

};
